# Support

- Use [GitHub Discussions](https://github.com/ShevasTest/playwright-guard/discussions) for usage questions and design ideas.
- Use [GitHub Issues](https://github.com/ShevasTest/playwright-guard/issues) for reproducible bugs and accepted feature work.
- Follow [SECURITY.md](SECURITY.md) for vulnerabilities.

Please include the Playwright Guard version, Playwright version, Node.js version, minimal configuration, and a small reproduction. Volunteer support has no guaranteed response time.
