## [0.1.0] - 2026-07-19

### Added

- Automatic Playwright fixture for console, page, request, and HTTP errors.
- String, regular-expression, and predicate allowlists.
- Deduplication, retention limits, and versioned JSON reports.
- Direct runtime API for advanced integrations.
- Strict TypeScript, unit and browser integration tests, CI, CodeQL, Dependabot, and Scorecard configuration.
